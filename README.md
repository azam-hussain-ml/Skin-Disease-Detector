# 🩺 AI Skin Disease Detector

An AI-powered web application that analyzes skin images and provides possible skin condition predictions using Deep Learning and Computer Vision.

Built with:
- 🤗 Transformers
- 🧠 Vision Transformer (ViT)
- 🎨 Gradio Web Interface
- 🐍 Python

---

# 🚀 Features

✅ Upload skin images instantly  
✅ AI-based image classification  
✅ Confidence score prediction  
✅ Professional medical-style report  
✅ Interactive Gradio web app  
✅ Beginner-friendly AI project  
✅ Public shareable link support  

---

# 📸 Project Demo

Upload a clear skin image and the AI will:

1. Analyze the image
2. Predict possible skin condition
3. Show confidence score
4. Provide medical-style advice

# 📷 Best Image Guidelines

For better AI predictions:

- Use clear images
- Use bright lighting
- Avoid blurry photos
- Keep skin area centered
- Use close-up skin photos
